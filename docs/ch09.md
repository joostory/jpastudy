JPA의 데이터 타입을 크게 분류하면 엔티티 타입과 값 타입으로 나눌 수 있다.
엔티티 타입은 @Entity로 정의하는 객체이고 값 타입은 단순히 값으로 사용하는 자바 기본 타입이나 객체

- 기본값 타입
  - 자바 기본 타입 (int, double)
  - 래퍼 클래스 (Integer)
  - String
- 임베디드 타입
- 컬렉션 값 타입

# 9.1 기본값 타입
```kotlin
@Entity
class Member(
  @Id @GeneratedValue
  var id: Long? = null,
  var name: String = "",
  var age: Int = 0
)
```
- Member는 엔티티
- id, name, age는 값: 공유 X
